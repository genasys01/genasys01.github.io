It's a single self-contained HTML file, so you can open it as-is or drop it on any web host. What's in it: - Dark CRT-style theme with neon purple/green glow, subtle scanlines and a flickering banner placeholder ready for your design. - Left nav with 6 sections; clicking one smooth-scrolls to and pulses its matching square panel on the right (Dashboard, Transmissions, Archive are open; Inbox, Profile, The Void are locked). - Top-right "LOG IN" opens a modal with Login/Sign Up tabs — passwords are SHA-256 hashed client-side before being stored, never in plain text. - Logging in (or signing up) unlocks the three locked panels and swaps the button for a username chip with a log-off option. One thing worth knowing: account data is saved in this artifact's **shared** storage, so it's visible to anyone using this page — fine for a demo/prototype, but swap in a real backend with proper auth before this handles real users or sensitive passwords.
